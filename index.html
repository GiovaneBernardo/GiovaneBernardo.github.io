Plaza Engine

•3D Game engine made with C++ and Vulkan, for general use, that gives the user complete control over the engine.
•The editor includes a node editor that can edit the render graph in real-time, without recompiling, making it easy and fast to change the pipeline.
•The default pipeline includes PBR + IBL, Deferred Rendering, Tiled Deferred Lighting, Cascaded Shadow Mapping with PCF, HDR, Bloom, Screen Space Reflections, ACES tonemapping.
•Shipped two playable games, made in just 3 days for Ludum Dare game jam.
•Scripting can use C++ or C#, while C# can only use the exposed api, C++ can interact with the whole engine.
•Technologies: Vulkan, GLSL, C++ and C# Scripting, PhysX 5, OpenAL, GLM, ImGui for editor GUI.
